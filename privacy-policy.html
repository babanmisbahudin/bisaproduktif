📋 Pendahuluan

BisaProduktif ("kami", "aplikasi") adalah aplikasi produktivitas gamified yang membantu pengguna membangun kebiasaan harian, mencapai tujuan, dan mendapatkan reward atas pencapaian mereka.

Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pribadi kamu saat menggunakan aplikasi BisaProduktif di perangkat Android.

📦 Data yang Kami Kumpulkan

Data yang kamu berikan langsung:

  • Nama pengguna
  • Nomor WhatsApp (opsional, hanya untuk klaim reward)
  • Alamat pengiriman (opsional, hanya untuk klaim reward fisik)
  • Data habit, goal, dan catatan yang kamu buat di dalam aplikasi

Data yang dikumpulkan otomatis:

  • Akun Google (nama dan email) jika kamu login dengan Google Sign-In
  • Data sesi fokus dan aktivitas produktivitas (untuk sinkronisasi)
  • Skor kepercayaan (trust score) untuk sistem anti-fraud reward

🎯 Cara Kami Menggunakan Data

  • Menyimpan dan menyinkronkan data produktivitas antar sesi
  • Memproses klaim reward dan menghubungi kamu via WhatsApp jika diperlukan
  • Menampilkan statistik dan progres pribadi kamu
  • Mendeteksi aktivitas tidak wajar untuk mencegah penyalahgunaan sistem reward
  • Mengirimkan notifikasi pengingat harian (hanya jika kamu mengaktifkannya)

☁️ Layanan Pihak Ketiga

Aplikasi ini menggunakan layanan berikut yang memiliki kebijakan privasi masing-masing:

  • Google Firebase (Authentication & Firestore) — penyimpanan data cloud dan login Google.
    firebase.google.com/support/privacy
  • Google Sign-In — login opsional menggunakan akun Google.
    policies.google.com/privacy
  • Google AdMob — iklan dalam aplikasi (jika ditampilkan).
    policies.google.com/privacy
  • OpenWeatherMap — data cuaca untuk tampilan home screen.
    openweathermap.org/privacy-policy
  • Google Gemini API — fitur AI Coach (jika diaktifkan pengguna).
    ai.google.dev/gemini-api/terms

💾 Penyimpanan Data

  • Data lokal — habit, goal, koin, dan pengaturan disimpan di perangkatmu menggunakan Hive (offline-first).
  • Data cloud — jika kamu login dengan Google, data akan disinkronkan ke Firebase Firestore untuk backup.
  • Penghapusan akun — kamu dapat menghapus data dengan menghubungi kami. Data cloud akan dihapus dalam 30 hari kerja.

🔒 Keamanan Data

Kami menggunakan enkripsi standar industri (HTTPS/TLS) untuk semua komunikasi dengan server. Data sensitif seperti nomor WhatsApp dan alamat hanya diakses oleh admin untuk keperluan pengiriman reward dan tidak dibagikan kepada pihak ketiga mana pun untuk tujuan pemasaran.

👶 Privasi Anak-anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak dengan sengaja mengumpulkan data pribadi dari anak di bawah 13 tahun. Jika kamu mengetahui bahwa seorang anak telah memberikan data pribadi kepada kami, harap hubungi kami segera.

✏️ Hak Pengguna

Kamu memiliki hak untuk:

  • Mengakses data pribadi yang kami simpan tentang kamu
  • Meminta koreksi data yang tidak akurat
  • Meminta penghapusan akun dan semua data terkait
  • Mencabut izin login Google kapan saja melalui pengaturan akun Google kamu
